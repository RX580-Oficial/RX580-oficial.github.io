Microsoft Flight Simulator 2020

Microsoft Flight Simulator 2020 es la última entrega de la legendaria serie de simuladores de vuelo, ofreciendo una experiencia de vuelo increíblemente realista y envolvente. Explora el mundo desde el cielo con aviones detallados, paisajes generados por IA y condiciones climáticas dinámicas. Ya seas un piloto experimentado o un novato, Flight Simulator 2020 te permite experimentar la emoción de volar como nunca antes.

REQUISITOS DEL SISTEMA

Mínimos:

Sistema Operativo: Windows 10 de 64 bits
Procesador: Intel Core i5-4460 o AMD Ryzen 3 1200
Memoria: 8 GB de RAM
Tarjeta gráfica: NVIDIA GeForce GTX 770 o AMD Radeon RX 570
DirectX: Versión 11
Almacenamiento: 150 GB de espacio disponible

Recomendados:

Sistema Operativo: Windows 10 de 64 bits
Procesador: Intel Core i5-8400 o AMD Ryzen 5 1500X
Memoria: 16 GB de RAM
Tarjeta gráfica: NVIDIA GeForce GTX 970 o AMD Radeon RX 590
DirectX: Versión 11
Almacenamiento: 150 GB de espacio disponible
